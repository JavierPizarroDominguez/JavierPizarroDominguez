## 🔭 Actualmente trabajando en

### 🏢 ERP Inmobiliario

Estoy desarrollando un ERP orientado a corredores y administradores de propiedades que busca simplificar la gestión completa de arriendos.

Entre sus funcionalidades se encuentran:

* 🏠 Administración de propiedades
* 📑 Gestión de contratos
* 💰 Control de cobros y pagos
* 🔧 Registro de reparaciones y mantenciones
* 📊 Gestión de comisiones
* 🧾 Control de gastos de servicios
* 🏦 Conciliación bancaria automática mediante cartolas

El objetivo es que el sistema pueda identificar automáticamente qué cobros han sido pagados y cuáles siguen pendientes a partir de los movimientos bancarios.

## 🛒 Plugin WooCommerce ↔ MercadoLibre

Desarrollé un plugin para WordPress en PHP puro que sincroniza inventario entre WooCommerce y MercadoLibre utilizando sus respectivas APIs.

### Tecnologías utilizadas

* 🐘 PHP
* 🛍️ API WooCommerce
* 📦 API MercadoLibre
* 🗄️ MySQL

El mayor desafío fue comprender y adaptar las múltiples reglas de negocio de MercadoLibre, especialmente en categorías, atributos obligatorios y guías de talla.
